# Internal Assistant

## 📂 Struktur Proyek

- **`internal_assistant_core.py`** → Mengatur konfigurasi client untuk pemanggilan API dan setup LangChain  
- **`internal_assistant_app.py`** → Menyediakan User Interface (UI) untuk menjalankan aplikasi  
- **`dependencies.py`** → Menyimpan daftar library pendukung  
- **`rag_modul/`** → Berisi logika untuk fitur Retrieval Augmented Generation (RAG)  
- **`progressProject/`** → Berisi logika untuk fitur Progress Project  
- **`requirements.txt`** → Menyimpan daftar dependency yang dibutuhkan agar kode dapat dijalankan  
- **`to_do_modul_test.py`** → Berisi pengujian untuk integrasi LLM dengan Microsoft To-Do
